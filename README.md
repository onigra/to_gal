## Installation

Add this line to your application's Gemfile:

    gem 'to_gal'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_gal

## Usage

```rb
require 'to_gal'

'ギャル文字に変換するよ'.to_gal #=> 'ギャル文字に変換するょ'
```
